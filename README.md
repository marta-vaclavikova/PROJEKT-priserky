## Monster code format

 **aabbccddeeff**

| Number | Body part |
| ----------- | ----------- | 
| aa | eyes |
| bb | body | 
| cc | legs |
| dd | arms |
| ee | mouth | 
| ff | colors | 

E.g. _010215052251_ is eys number 01, body 02, legs 15, arms 05, mouth 22, colors 51
